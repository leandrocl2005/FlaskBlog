Flask para Udacity

Este é um projeto em flask feito como pré-requisito para o Nanodegree de desenvolver Full-Stackda Udacity.

Nesse projeto foram avaliados os seguintes requisitos:

  • O projeto implementa um JSON endpoint que exibe as mesmas informações exibidas em HTML endpoints de um item qualquer do catálogo.
  • O site lê as informações de item e catálogo de um banco de dados.
  • Criação de CRUD para criar, ler, atualizar e deletar posts.
  • A página implementa um serviço de autorização e autenticação de terceiros (como Google Accounts ou Mozilla Persona), em vez de implementar suas próprias especificações de autenticação e autorização.
  • Certifique-se de que o projeto inclua um botão/link de "login" e "logout". A aparência dele depende da opção do aluno.

Para acessar o endpoint entre em http://localhost:5000/post/<post_id>/getJson

post_id é o id do post que deseja consultar o Json
